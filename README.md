# springfox-plus

## Introduction

springfox-plus是一个基于springfox3二开的在线文档工具，其核心设计目标是让springfox3更好用。

## Features
- 文档中的入参，支持按照java类中属性位置展示
- 支持入参属性分组；同一个类被不同场景使用
